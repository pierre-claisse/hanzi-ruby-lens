## Changes Summary

### TitleBar.tsx
- Add `onAddText` and `showAddButton` props
- Render add button (Plus icon) before PaletteSelector, visible only when `showAddButton` is true (library view)
- Add centered text title element (visible only in reading view), with `textTitle` prop
- Move zoom indicator `({zoomLevel}%)` visibility to reading view only (alongside zoom buttons, behind `showBack`)
- Increase app title `<h1>` font from `text-sm` to `text-base`
- Title bar layout: left group (back button + app title) | center (text title, absolute positioned) | right group (buttons)

### LibraryScreen.tsx
- Remove the FAB button entirely
- Remove the `onAddText` prop (add button moves to TitleBar)
- Change preview list from `flex flex-col gap-2` to CSS grid: `grid grid-cols-[repeat(auto-fill,minmax(240px,1fr))] gap-4`

### TextPreviewCard.tsx
- Increase title `<h3>` font from `text-base` to `text-lg`
- Remove `w-full` since cards are now sized by grid

### App.tsx
- Remove `<h2>` title heading from reading view content area
- Pass `onAddText={handleAddText}` and `showAddButton={appView === "library"}` to TitleBar
- Pass `textTitle={activeText?.title}` to TitleBar for centered display in reading view
- Remove `onAddText` prop from LibraryScreen

### Test Updates
- Update tests that assert add button location (now in title bar across all views where `showAddButton` is true)
- Remove assertions for title heading in reading view content area
